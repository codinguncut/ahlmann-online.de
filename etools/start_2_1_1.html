Zuordnungsübungen

Beispiel: "American and British English", Englisch-Oberstufe

Beispiel: "Activities", Englisch-Grundstufe, mit Bildern

Beispiel: "Nebensätze", DaF-Grundstufe, Satzteile zuordnen

Per drag & drop wird immer ein Element rechts dem passenden Element links zugeordnet.

Falsche Zuordnungen werden akzeptiert, die Lerner können aber jederzeit ihre Lösungen kontrollieren und dann weiterüben.

JMatch verlangt immer eine 1:1-Zuordnung - das heißt, Übungen, die eine Zuordnung von Elementen zu Kategorien erwarten, lassen sich damit nicht herstellen (z.B. Tee, Wein, Kaffee etc in die Kategorie "Getränke" einsortieren und Butter, Käse, Joghurt zu "Milchprodukte" - das wäre eine 1:n-Zurdnung und funktioniert nicht)

  

Lückentexte, Cloze-Tests

Beispiel: "An Occurence at Owl Creek Bridge" von Ambrose Bierce, Quelle: Project Gutenberg

 

Mit "JCloze" lassen sich verschiedene Vairanten von Lückentexten realisieren (auch als Clozetest bekannt).

Man kann wahlweise die fehlenden Wörter in der Übung angeben, Hinweise schreiben (wie bei einem Kreuzworträtsel) oder Einzelbuchstaben als Hinweise zulassen. Die Übung - eigentlich eine Testform - ist also immer lösbar.

JCloze hat eine "AutoGap"-Funktion - der klassische Cloze-Test entfernt jedes n-te Wort aus einem Text - aber in der Regel wird die Autorin oder der Autor die Lücken selbst festlegen wollen, um bestimmte Vokablen, Wortklassen etc. zu üben. Das geht natürlich mit JCloze auch.

  

Freie Texteingabe mit JQuiz

Beispiel: Abbreviations, Englisch-Oberstufe

 

Gefordert wird eine konkrete Antwort auf eine Frage.

Vorsicht: Diese Aufgabenform ist grundsätzlich sehr problematisch. Der Computer kann plausible, sogar richtige, Antworten nicht erkennen (Schreibvarianten, Zahlen statt Buchstaben u.u., kleine Tippfehler, alternative kluge Lösungen), wenn Sie beim Erstellen der Aufgabe nicht daran gedacht haben, konkret diese Antwort als mögliche Lösung anzugeben. Das ärgert die Lerner ziemlich massiv. Bei JQuiz können Sie immerhin beliebig viele Varianten, einen Hinweis- und einen Lösungsbutton integrieren, so dass die Aufgabe zumindest lösbar bleibt. Trotzdem: Computer können nicht denken!!!!!!!!!!!!!!!
