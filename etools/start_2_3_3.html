Den Ausdruck auf Papier kann man beim Crossword Compiler weitgehend selbst steuern.

Bei Webseiten sieht es etwas anders aus: man kann Text- und Hintergrundfarbe, Aufschriften und Schriftart der Buttons etc. über ein Dialogfeld einstellen. Wer ernsthaft in die Seite hineingreifen möchte, sollte sattelfest in HTML und Java sein, dann kann man nicht nur das Layout, sondern auch die Funktionalität anpassen.

Preis der Freiheit: Selbstverständlich kann man sich so das Design und die Funktion der Seite auch komplett zerschießen. Keine Chancen ohne Risiken!
