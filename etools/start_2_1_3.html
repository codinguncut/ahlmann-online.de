Farbe, Schrift und Aufschrift der Buttons kann man in allen Hot Potatoes über Dialogfenster einstellen. So kann man die Übung optisch an die eigene Website anpassen.
Wer HTML und Java kann, kann sich darüber hinaus praktisch unbegrenzt austoben.

Als Farben stehen die Webfarben zur Verfügung. Schriften müssen wie üblich auf dem Rechner vorliegen, damit man sie auswählen kann.

Wenn Sie die Übungen einbetten in z.B. eine Website, sollten Sie hier die Schrift auswählen, die Sie auch auf der Website verwendet haben. Bedenken Sie, daß die Schrift webtauglich sein sollte.

Es ist bei der Farbwahl zu bedenken, daß man Schrift auf Hintergrund lesen können sollte.
